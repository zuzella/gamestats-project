# GameStats ⚽

GameStats is a football match winner prediction app built with Python and Streamlit.

## Features
- Select two national teams
- Analyze historical head-to-head matches
- View wins, draws, and outcome distribution
- Get a simple prediction with estimated confidence

## Tech Stack
- Python
- Pandas
- Streamlit
- scikit-learn

## Dataset
International football results dataset from Kaggle.

## Run locally
pip3 install -r requirements.txt
streamlit run app.py
